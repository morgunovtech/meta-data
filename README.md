# Meta Data Insight

Meta Data Insight — это одностраничное приложение на React, которое помогает оценить приватность фотографии: оно извлекает и визуализирует метаданные, строит карту, проверяет погоду и объекты поблизости, а также даёт советы по очистке и выгрузке снимка без лишних данных. Все тяжёлые вычисления выполняются в браузере; Pages Functions используются только как прокси к открытым API.

## Ключевые возможности

- Полностью локальная обработка изображений: извлечение/очистка EXIF/XMP/IPTC/ICC, размывание лиц и ручные маски, анти-поиск и защита от PRNU.
- OCR в браузере с fallback-ассетами и препроцессингом для стабильного распознавания текста и выделения областей.
- Подсказки по приватности: пресеты сценариев публикации, скоринг рисков и автоматические рекомендации «сделать безопаснее».
- Работа с локацией и временем: карта, погода, определение часового пояса и безопасное обращение с координатами.
- Предпросмотр «до/после», отчёт об изменениях и проверяемая очистка перед скачиванием.

## Jobs To Be Done

- «Я хочу быстро понять, что фото раскрывает о месте/времени/устройстве, не передавая файл в сеть».
- «Мне нужно замазать лица, тексты или номера и скачать безопасную версию одним кликом».
- «Хочу убедиться, что метаданные удалены, а итоговый файл не содержит лишних утечек».
- «Нужно подготовить фото для соцсетей/работы/доказательств с подходящими настройками по умолчанию».

## Локальная разработка

```bash
npm install
npm run prepare:ocr
npm run dev
```

Приложение будет доступно на http://localhost:5173.

Скрипт `npm run prepare:ocr` копирует ассеты Tesseract (worker/wasm/eng.traineddata) из `node_modules` в `public/ocr`, чтобы OCR работал без доступа к CDN.

Для быстрой проверки OCR в dev-окружении есть smoke-команда `npm run smoke:ocr`, которая строит простую PBM-картинку с текстом `TEST 123` и убеждается, что движок распознаёт оба токена.

Во время разработки Vite автоматически подключает Pages Functions из каталога `functions/` — HTTP-запросы на `/api/*`
выполняются тем же кодом, что и в продакшене, поэтому блок «Что можно узнать из данных» доступен без дополнительных настроек.

## Сборка

```bash
npm run build
```

Готовые статические файлы окажутся в каталоге `dist/`.

## Деплой на Cloudflare Pages

- **Build command**: `npm run build`
- **Build output directory**: `dist`
- **Functions directory**: `/functions`

Pages Functions предоставляют прокси для:

- Nominatim (обратное геокодирование);
- Open-Meteo (часовой пояс и погодный срез);
- Date.Nager (праздники);
- Overpass API (POI и объекты наблюдения).

Каждая функция валидирует входные параметры и нормализует ответ в компактный JSON (`{ ok, data | error }`).

## Безопасность и приватность

- Изображения не покидают браузер.
- Метаданные и карты отображаются локально; на Functions отправляются только координаты, даты и прочие примитивы.
- Очистка метаданных и размытие выполняются через `<canvas>` полностью на клиенте.
